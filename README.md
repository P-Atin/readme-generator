# 🚀 README Generator

Un outil moderne et intuitif pour générer des fichiers README professionnels en quelques secondes. Construit avec **React**, ce projet met l'accent sur l'expérience utilisateur (UX) et la performance technique.

---

## 📸 Aperçu du Projet

L'application offre une interface "Split-Screen" permettant de modifier les informations à gauche et de voir le rendu Markdown formaté instantanément à droite.

> **Lien du projet en ligne :**

---

## ✨ Fonctionnalités Clés

- **⚡ Live Preview** : Visualisation instantanée des changements au format Markdown grâce à `react-markdown`.
- **💾 Persistance des données** : Utilisation du `localStorage` pour ne jamais perdre votre progression, même après avoir fermé l'onglet.
- **🛡️ Badges Dynamiques** : Génération automatique de badges GitHub (License, Stars, Issues) basés sur votre pseudo et nom de dépôt.
- **📋 Copie en un clic** : Intégration de l'API Clipboard pour copier le texte brut instantanément.
- **📥 Téléchargement Direct** : Génération d'un fichier `.md` téléchargeable via les API Blob du navigateur.
- **🧹 Système de Reset** : Bouton de réinitialisation sécurisé pour vider le formulaire et le cache.

---

## 🛠 Tech Stack

- **Framework** : [React.js](https://reactjs.org/) (Vite)
- **Styling** : CSS3 Custom (Design SaaS moderne et responsive)
- **Markdown Rendering** : `react-markdown`
- **Notifications** : `react-hot-toast`
- **Icons** : `lucide-react`
